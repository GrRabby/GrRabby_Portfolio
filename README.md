A clean, responsive personal portfolio website built with pure HTML and CSS, showcasing skills, experience, and projects of a full-stack software engineer specializing in MERN development, Python automation, and mobile/desktop applications.

## ⚙️ Getting Started

No installation or build step needed. Just clone and open in a browser.

```bash
# Clone the repository
git clone https://github.com/GrRabby/GrRabby_Portfolio.git

# Navigate into the project
cd GrRabby_Portfolio

# Open in browser
open index.html
# or simply double-click index.html
```

---

## 🚀 Deployment

### GitHub Pages
1. Go to **Settings → Pages** in your repository.
2. Set the source branch to `main` and folder to `/ (root)`.
3. Save — your site will be live at `https://grrabby.github.io/GrRabby_Portfolio/`.

### Vercel
1. Import the repository at [vercel.com](https://vercel.com/).
2. No build configuration needed — deploy as a static site.
